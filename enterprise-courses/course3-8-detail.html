📋 课程总览

本课程旨在通过钉钉“悟空”平台，帮助高管构建一套“感知-行动-进化”的自动化销售飞轮，将AI从“聊天工具”升级为企业的“第二管理层”。课程为期2天（每天≥6.5小时），采用20%认知导入 + 50%场景Demo演示 + 30%现场Workshop共创的模式。

🚀 核心学习目标

  • 认知重塑：理清生成式AI与智能体（Agent）的本质区别，确立悟空作为业务执行者的战略定位。
  • 掌握三大核心场景：开源（AI全域雷达）、管控（AI执行督导官）、进化（AI经营复盘官）。
  • 落地实战：现场设计并配置一个专属岗位的“硅基助手”，带走可执行的AI销售增长方案。

📖 详细课程大纲

模块一：认知重塑 —— 从“聊天”到“执行”的范式转移

  • AI祛魅与定义：清晰界定“生成式AI”与“智能体 Agent”的本质区别。
  • 悟空平台的定位：明确企业需要的不是“会说话的AI”，而是能深度连接钉钉生态的“第二管理层”。
  • 互动 Demo 演示（同题异构挑战）：现场对比通用大模型 vs 悟空平台在处理业务指令时的差异。

模块二：实战场景一 —— AI全域雷达（开源·感知层）

  • 核心理念：让每个销售都成为一家“流量公司”，利用新媒体渠道（小红书、公众号等）拓展客源。
  • 角色重塑：销售 + 悟空 = 市场（内容官）+ 销售（联络官）+ 售前（方案官）。
  • 落地 Demo 演示：小红书“种草-转化”全链路闭环（内容裂变 -> 线索捕获 -> 自动触达）。

模块三：实战场景二 —— AI执行督导官（管控·行动层）

  • 核心痛点：解决销售过程“黑盒化”问题，从“事后看结果”转向“实时管过程”。
  • 悟空解决方案：节点自动打卡 + 异常实时预警 + 话术即时纠偏。
  • 落地 Demo 演示：响应时效督导、话术合规纠偏、停滞线索唤醒。

模块四：实战场景三 —— AI经营复盘官（进化·战略层）

  • 核心痛点：传统复盘滞后、主观、碎片化，无法指导即时行动。
  • 悟空解决方案：数据-洞察-行动闭环。自动归因分析，生成策略并下发任务。
  • 落地 Demo 演示：从“看数据”到“给策略”（异常预警 -> 深度归因 -> 策略生成 -> 任务分发）。

模块五：Workshop智能体共创 —— 打造岗位的“硅基助手”

  • 共创规则说明：铁三角组队、痛点挖掘、画布填写。
  • 悟空装机与基础操作：手把手教学创建智能体、上传知识库、配置触发器。
  • 现场共创与辅导：各组实战配置专属“硅基助手”，跑通最小闭环（MVP）。
  • 成果路演与点评：小组展示、导师点评、颁奖与总结。

🎯 Demo场景：AI全域雷达

场景目标：赋能销售从“资源接收者”跃迁为“超级个体”，通过悟空实现“能力内化”，让每个销售个体都具备一个“微型市场部”的感知与运营能力。

第一步：跨渠道多模态感知（悟空在“看”与“听”）

悟空自动调用浏览器工具，深入指定平台的垂直领域话题页。不仅抓取文字，更通过多模态能力“看懂”视频中的设备型号、“听懂”博主提到的产能瓶颈，并从数百条最新评论中筛选出高意向关键词。

第二步：智能分级与画像补全（悟空在“想”）

悟空根据预设的 B 端客户特征，将线索自动标记为 S/A/B 级，并关联该客户的历史公开动态，补全其行业属性、企业规模等背景信息，形成初步的《客户微画像》。

第三步：合规钩子生成与任务投喂（悟空在“做”）

针对 S 级线索，悟空拒绝生成硬广，而是基于“利他原则”自动生成价值钩子。最终在钉钉上向销售推送一张结构化的《线索卡片》，销售只需点击“一键复制”即可进行合规、专业的私信触达。

💡 量化价值
  • • 效率跃迁：将原本需要数天的人工搜集与甄别工作缩短至5分钟，线索获取效率提升 90% 以上。
  • • 质量提升：通过意图识别过滤掉 80% 的无效噪音，确保销售接触的每一条线索都具备高转化潜力。
  • • 合规保障：100% 规避因违规引流导致的封号风险，保护企业数字资产安全。
